Chapter 1: Variables

1.1: Declare Variables

In C#, variables are used to store data during program execution. Every variable in C# must be declared before use and has a specific data type.

type name;

A data type specifies the type of data that a variable can store. Examples: int, double, string.

A variable name is the identifier used to store and reference a value. Example: age, userName.

For example:

int age;
string name;
float weight;
bool isMale;

After you declare the variable, you can assign a value using the equals sign.

For example:

age = 20;
weight = 60.5f;
name = "John Doe";
isMale = true;

Or you can declare and assign the value for the variable in a single line.

int age = 20;
float weight = 60.5f;
string name = "John Doe";
bool isMale = true;

Also, C# supports the temporary variables or local variables. Temporary variables are declared in a block of code. But you can turn a local variable to a global variable by typing public keyword (for global variables). When the block of code ends, temporary variables are discared and can't be used after.

For example:

public class myClass {
int x = 5;
public int y = 10;
}
Console.WriteLine(y);
Console.WriteLine(x); Error CS0103: The name 'x' does not exist in the current context

Also, in C# has special kinds of variables like constants and static variables, is used to store values that don't change. A variable can be this type by typing the const keyword (for constants) and the static keyword (for static variables).

const int x = 10;
x = 10; Error CS0131: The left-hand side of an assignment must be a variable, property or indexer

1.2: Data Types

In C#, data type is an important concept, allowing us to declare variables and functions with a specific type of data. Data type also allows the program to use that data to do maths, evaluate datas... C# supports many types of datas including:

  • Natural numbers (integer): int, long, short, byte, sbyte, uint, ulong, ushort
  • Real numbers (floating-point): float, double, decimal
  • Character-type (character): char
  • String-type (string): string
  • Logic-type (boolean): bool
  • Object-type (object): object
  • No value (null value): null

    • Every type of data has their own behaviors. Like integers (ℤ) can only store natural numbers (ℕ) that is a positive number. On the other side, real numbers-type (ℝ) is used to store decimals and complex numbers (ℂ).

    Along with the basic data types above, C# lets us create new and more advanced data types with struct & class from basic data types.

    Integers
    int a = 10;
    long b = 1000000;
    byte c = 255;
    Floats
    float x = 3.14f;
    double y = 3.141592653589793;
    decimal z = 100.5m;
    Character
    char ch = 'A';
    String
    string str = "Hello!";
    Booleans
    bool isTrue = true;
    Objects
    object obj = new Object();
    Null
    int? nullableint = null;
    int unnullableint = null; Error CS0037: Cannot convert null to 'int' because it is a non-nullable value type

    1.3: Maths in C#

    There are symbols in C# that are used to indicate expressions and evaluation. These are all the maths symbols in C#:

    • Numbers: addition (+), subtraction (-), multiplication (*), division (/), modulo (%)
    • Assignments: assign (=), add and assign (+=), subtract and assign (-=), multiply and assign (*=), divide and assign (/=), mod and assign (%=)
    • Compare: equals (==), smaller (<), bigger (>), smaller or equal to(<=), bigger or equal to
    • Logic: and (&&), or (||), negative (!)
    • Increment or decrement by one: increment by one (++), decrement by one (--)

    For example:

    int a = 10 + 5; Returns 15.
    int b = 10 - 5; Returns 5.
    int c = 10 * 5; Returns 50.
    int d = 10 / 5; Returns 2.
    int e = 10 % 3; Returns 1.
    int f = 10;
    f += 5; Returns 15.
    f -= 5; Returns 10.
    f *= 2; Returns 20.
    f /= 2; Returns 10.
    f %= 4; Returns 2.
    bool g = (10 == 5); Returns false.
    bool h = (10 != 5); Returns true.
    bool i = (10 < 5); Returns false.
    bool j = (10 > 5); Returns true.
    bool k = (10 <= 5); Returns false.
    bool l = (10 >= 5); Returns true.
    bool m = true;
    bool n = false;
    bool o = (m && n); Returns false. To get true here, m & n has to both be true.
    bool p = (m || n); Returns true. To get true here, only one variable has to be true.
    bool q = !m; Returns false. The value will be reversed.
    int x = 100;
    int y = 200;
    int z = (10 > 5) ? x : y; Returns 100. If true, returns first value. If false, return second value.
